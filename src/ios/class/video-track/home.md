# VideoTrack

```swift
public class VideoTrack : MediaTrack {

    public var width: Int! { get }

    public var height: Int! { get }
}
```

# 속성

## width

| 타입 | 설명 | 설정 |
|:--:|:--:|:--:|
|Int!|가로 길이 (픽셀)|불가능|

가로 길이를 나타내는 속성입니다. 단위는 픽셀입니다.

## sampleRate

| 타입 | 설명 | 설정 |
|:--:|:--:|:--:|
|Int!|세로 길이 (픽셀)|불가능|

새로 길이를 나타내는 속성입니다. 단위는 픽셀입니다.
